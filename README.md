# SpartaMarket_DRF

## 주제

기존에 만든 sparta market은 중고거래 중개 사이트의 백앤드 부분만 따로 DRF를 사용해서 작성했다.
회원가입, 로그인, 게시물 등록등의 기능을 Django REST Framework를 통해 API를 JSON형태로 
응답하는 서버를 만들었다.

***

## 개발 기간 
 13.04.24 ~ 16.04.24 개인 일정이있 약 4일간 진행했습니다

***

## 필수 구현 기능 
  1. 회원가입 기능
  2. 로그인
  3. 프로필 조회
  4. 상품 등록
  5. 상품 목록 조회
  6. 상품 수정
  7. 상품 삭제
